# Arana UI

This project is initialized with [Ant Design Pro](https://pro.ant.design). Follow is the quick guide for how to use.

## Environment Prepare

Install [Node](https://nodejs.org/en/)

Install `node_modules`:

```bash
npm install
```

### Mock Server

npm install -g @stoplight/prism-cli

```bash
npm run mock
```

### Start project

```bash
npm start
```

### Build project

```bash
npm run build
```
